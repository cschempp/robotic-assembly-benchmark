CAD part and assembly files


→ cad_prt_asm.zip
This includes all parts and assembly groups in Creo native file formats .prt and .asm. The predefined assembly groups are described in the following table.

→ cad_step.zip
This includes all parts in .step file format.

→ cad_stl.zip
This includes all parts in .stl file format.

For downloading individual files, see the images below. The link to download the respective file are provided beneath each image. Depending on your browser, you may have to right-click, "save target as"/"save link as" to save the file.

benchmark_assembly.asm:
The completely assembled benchmark, without the board and fixations.

benchmark_start_position.asm:
The start position of the benchmark, with all components aranged at their initial positions.

benchmark_final_position.asm:
The final state of the benchmark, when all components are assembled.

housing_bottom_assembly.asm:
Assembly group of all components which have to be assembled into the bottom housing.

housing_middle_assembly.asm:
Assembly group of all components which have to be assembled into the middle housing.

housing_middle_assembly_initial.asm:
The middle housing assembly with only the fixed parts.



housing_top_assembly.asm:
Assembly group of all components which have to be assembled into the top housing.


housing_top_assembly_initial.asm:
The top housing assembly with only the fixed parts.
